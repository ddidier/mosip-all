# inji verify services

## Install
* Run `install-all.sh` to deploy inji verify services.
  ```
  cd deploy
  ./install-all.sh
  ```

## Delete
* Run `delete-all.sh` to remove inji verify services.
  ```
  cd deploy
  ./delete-all.sh
  ```

## Restart
* Run `restart-all.sh` to restart inji verify services.
  ```
  cd deploy
  ./restart.sh
  ```
